# MAXGRIFOS V2 DESIGN SYSTEM

## Estética: Neon Flex Minimalista
- **Fondo Dominante**: Blanco puro (#FFFFFF).
- **Headers / Barras**: Azul Brillante / Electric Blue (#0066FF).
- **Acentos**:
  - Neon Cyan (#00FFFF) para estados activos sutiles.
  - Verde Éxito (#10B981).
  - Naranja Alerta (#F59E0B).
  - Rojo Crítico (#EF4444).

## Componentes
- **Botones**: Claros, con bordes suaves, microinteracciones (escala 0.98 al presionar).
- **Cards**: Fondo blanco, sombra muy tenue (shadow-sm), borde suave.
- **Tipografía**: Sans-serif moderna, legible, pesos consistentes.
- **Touch Targets**: Mínimo 44px x 44px para movilidad.

## Restricciones
- PROHIBIDO: Fondo negro dominante por defecto.
- PROHIBIDO: Saturación excesiva.
- PROHIBIDO: Estética enterprise antigua (bordes duros, colores apagados).
