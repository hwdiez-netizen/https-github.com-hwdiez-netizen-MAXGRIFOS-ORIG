# MAXGRIFOS V2 MODULE REGISTRY

## Módulos Críticos (Prioridad 1)
- **Maestro Productos**: Gestión de SKUs, Atributos y Precios.
- **Kardex / Inventario**: Movimientos de stock y valoración.
- **Clientes / Proveedores**: Directorio maestro.
- **Auditoría**: Ledger de trazabilidad.

## Módulos Operativos (Prioridad 2)
- **Compras**: Órdenes y Recepción.
- **Ventas / Pedidos**: Captura de demanda.
- **Facturación**: Cierre transaccional.
- **Garantías**: Gestión de post-venta.

## Módulos Estratégicos (Prioridad 3)
- **Cartera / Recaudos**: Gestión financiera.
- **KPIs**: Dashboards comerciales.
- **Tesorería**: Egresos y Flujo de caja.

## Motores de Protección (Keep from V1)
- **SKU Engine**: Generación de códigos únicos.
- **Scanner Core**: Decodificación de Code128 y QR.
