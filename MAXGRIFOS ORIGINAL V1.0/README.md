# MAXGRIFOS ERP

PWA mobile-first construida con Vite para validación local, GitHub y Vercel.

## Comandos locales

npm install
npm run build
npm run dev -- --host 0.0.0.0

## Vercel

Framework: Vite
Build Command: npm run build
Output Directory: dist

## Seguridad

No subir .env, secretos, tokens ni credenciales.
Validar persistencia de datos offline antes de promover a producción.
