# HOME DASHBOARD TARGET IMAGE REFERENCE — MAXGRIFOS ERP V2

## Estado oficial

Esta carpeta contiene la referencia visual oficial del Home Dashboard premium aprobado por el usuario.

La imagen target debe ser insertada manualmente por el usuario humano en esta ruta:

`docs/reference/HOME_DASHBOARD_MAXGRIFOS.png`

## Regla principal

THIS IS THE TARGET DESIGN.
DO NOT CREATE A NEW DESIGN.
DO NOT REINTERPRET.
PRESERVE TWO MODULE CARDS PER ROW.
PRESERVE 3D PREMIUM BUTTONS.
PRESERVE APPLE-LIKE MOBILE FLOW.

DO NOT SIMPLIFY THE DESIGN.
DO NOT CONVERT MODULE CARDS TO ONE COLUMN BY DEFAULT.
DO NOT REMOVE PREMIUM DEPTH.

## Uso permitido

Esta imagen solo puede usarse como referencia visual para preservar:

- layout premium del Home Dashboard;
- dos tarjetas por fila en módulos principales;
- tarjetas brillantes con efecto 3D;
- íconos grandes y coloridos;
- KPI strip premium;
- hero card premium;
- bottom navigation premium;
- estilo Apple-like mobile-first;
- jerarquía visual limpia;
- ausencia de FAB central;
- ausencia de botón central destacado;
- ausencia de doble navegación inferior.

## Uso prohibido

Está prohibido:

- copiar arquitectura externa;
- copiar lógica externa;
- copiar persistencia externa;
- copiar legacy visual;
- inventar un diseño alterno;
- reemplazar este target por una versión simplificada;
- transformar el Home Dashboard en una sola columna por defecto;
- remover profundidad visual;
- remover botones/tarjetas 3D;
- introducir bottom nav con botón central destacado;
- introducir FAB invasivo;
- introducir paneles técnicos NIS visibles.
